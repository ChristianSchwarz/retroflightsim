Help

Controls

Keyboard

  • W/S: Pitch
  • A/D: Roll
  • Q/E: Yaw
  • Z/X: Throttle
  • Num Lock: Telemetry graph

Joystick

No device detected
  • Axis 1: Pitch
  • Axis 0: Roll
  • Axis 3: Yaw
  • Axis 2: Throttle

Systems

  • G: Landing gear
  • F: Flaps
  • L: FCS limiters (AoA/g) on/off
  • T: Select target
  • I: Target night view
  • H: Cycle HUD focus
  • R: Flight recorder (toggle, downloads JSON)
  • F10: Import aircraft mod (.zip)

Spawn menu

  • Esc: Open spawn menu
  • 1: Approach spawn
  • 2: Runway spawn
  • Aircraft list: pick from the combobox in the spawn menu

Views

  • N: Day/night
  • F1: Cockpit
  • F2: Exterior back/front
  • F3: Exterior left/right
  • F6: Cycle static aircraft models
  • F12: Aircraft showcase (black background, orbit with numpad)
  • 4: To/from target
  • Numpad 4/6/8/2: Move camera around aircraft
  • Numpad 5: Recenter camera
  • Numpad * / /: Zoom in / out
